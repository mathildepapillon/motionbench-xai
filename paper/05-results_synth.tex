\title{Synthetic Benchmark Results\\[0.4em]
\large EC1\,/\,EC2\,/\,EC3 across imputers, datasets, and groupings}
\author{}
\date{}
\maketitle

\noindent
Each table reports \textbf{mean~$\pm$ std} across $n$ held-out test sequences.
All three metrics are lower-is-better~($\downarrow$):
\begin{itemize}\itemsep2pt
  \item \textbf{EC1}~=~mean absolute error $\frac{1}{K}\sum_k |\hat\phi_k - \phi^*_k|$
  \item \textbf{EC2}~=~mean squared error over all sampled coalitions
  \item \textbf{EC3}~=~$1 - \text{Pearson}(\hat\phi, \phi^*)$
\end{itemize}
\textbf{Bold} marks the best non-oracle method per column.
The Gaussian Oracle rows are separated below the rule and serve as an upper bound
(they use exact Gaussian conditionals; EC3~$>$~0 because the KernelSHAP estimator
itself introduces finite-sample noise).

\bigskip

\section{Gaussian Temporal}
%% ============================================================

\noindent
Data: equicorrelated-joint, AR(1)-temporal synthetic Gaussian motion
($K$ temporal windows, $J=17$ joints, $F=3$ features).
Classifier: small MLP with BatchNorm.
Truth key: \texttt{gaussian\_oracle} (exact Gaussian conditional).
$K_\mathrm{MC}^\mathrm{true}=1000$ coalitions, 20 completion samples per coalition.
